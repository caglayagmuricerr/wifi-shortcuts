# wifi-shortcuts
Python script for Windows 11 to connect and disconnect from a specified WiFi network using keyboard shortcuts for faster WiFi management.

## HOTKEYS
`CTRL + ALT + C :` Connect <br>
`CTRL + ALT + D :` Disconnect
